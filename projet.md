Ce repository doit contenir les fichiers de création d'une image docker, dans laquelle on fait tourner une wordpress7. 
L'image doit embarquer mysql, apache et php.
trois repertoires doivent exister en dehors de l'image : les fichiers de la base de données doit être dans un repertoire externe ,  le repertoire wp-content et wp-plugins.
Ces trois repertoires seront , pour les tests, localiser à la racine du repository
